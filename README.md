# MySQL-SetUp-Guid
Installation of my SQL in Windows 11

# MySQL Installation Guide (Windows 11)

## Step 1: Download MySQL

Go to: https://dev.mysql.com/downloads/installer/

## Step 2: Run Installer

* Choose: Developer Default
* Click Next → Install

## Step 3: Configuration

* Set root password
* Keep default port: 3306

## Step 4: Finish Setup

* Click Execute → Finish

## Step 5: Verify Installation

Open Command Prompt:

```
mysql -u root -p
```

Enter password → MySQL should start

## Done 🎉

# MySQL Installation Guide for (Mac OS)

## 🍎 macOS Installation

### Step 1: Download MySQL

Go to:
https://dev.mysql.com/downloads/mysql/

* Select: macOS
* Download `.dmg` file

---

### Step 2: Install MySQL

* Open the downloaded `.dmg` file
* Run the MySQL installer package
* Follow the instructions

---

### Step 3: Set Root Password

* During installation, set your root password
* Save it safely ⚠️

---

### Step 4: Start MySQL Server

Go to:
System Settings → MySQL → Start MySQL Server

---

### Step 5: Verify Installation

Open Terminal and run:

```bash id="k92hd7"
mysql -u root -p
```

Enter your password → MySQL should start

---

## ✅ Done 🎉
